Zuno Chat is a full-stack AI-powered chatbot application that enables intelligent conversational interactions. It includes a frontend client and backend services that together deliver real-time chat experiences, user authentication, and scalable deployment.
Live Demo: https://zuno-chat.vercel.app

Features:
Interactive interface
Backend APIs supporting chat logic
User authentication
Fully responsive frontend
Easy deployment via Vercel and railway backend
Clean project structure for frontend and backend separation

Tech Stack:
Frontend: React / Next.js / JavaScript
Backend: Node.js / Express
Deployment: Vercel / serverless environment
Real-time interactions and API handling
